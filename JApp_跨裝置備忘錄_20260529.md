## 三、整體架構說明

### 後端（app.py）
```
Api 類別
├── get_videos()                  → 讀取 Favorites.json
├── save_videos()                 → 寫入 Favorites.json
├── fetch_video_info()            → 多平台並發查詢番號資訊
├── search_all_platforms()        → 跨平台搜尋（JAVxx + TCAV + TKTube）
├── get_tktube_home_videos()      → TKTube 首頁/分頁爬蟲
├── get_home_videos()             → JAVxx 首頁爬蟲
├── get_tcav_home_videos()        → TCAV 首頁爬蟲
├── scan_pages_for_date()         → 二分搜尋定位目標日期頁碼
├── get_embed_url()               → 取得影片嵌入播放 URL
├── heal_favorites_background()   → 背景自動修復最愛資料
├── init_playwright_background()  → 啟動無頭瀏覽器常駐工作佇列
└── copy_to_clipboard()           → Windows 剪貼簿操作
```

### 前端（script.js）主要模組
```
homeManager（IIFE）
├── preload()           → 程式啟動後 500ms 背景預載首頁
├── onHomeActivated()   → 切換到首頁分頁時觸發，瞬間渲染
├── fetchPage()         → 載入指定頁碼的 TKTube 影片
├── doSearch()          → 全平台跨站搜尋
├── doDateFilter()      → 日期篩選 + 後端二分法定位
├── initDateFilters()   → 三級聯動下拉選單初始化
└── bindEvents()        → 所有按鈕事件綁定

createVideoCardElement(v, source)
├── source = 'home' → 顯示「新增至最愛」右鍵選單
└── source = 'fav'  → 顯示「移除最愛」右鍵選單 + hover 時複製番號到剪貼簿

addToSyncWatch(url)   → 加入同步觀看欄位
setupPaneControls()   → 同步觀看面板設定（含 A/B 點迴圈播放）
```

### 四大分頁

## 五、重要技術細節

### 爬蟲架構
- 全部使用 urllib.request 直接爬取（速度約 200ms/頁）
- 禁止在主要 API 執行緒呼叫 Playwright 同步 API（會導致 greenlet 跨執行緒例外）
- Playwright 僅用於常駐背景工作佇列（處理特定需要 JS 渲染的頁面）

### TKTube 分頁 URL 格式
```
第 1 頁：https://tktube.com/zh/latest-updates/
第 N 頁：https://tktube.com/zh/latest-updates/{N}/
```
注意：不是 ?page=N，是斜線格式

### 日期欄位對應
- TKTube 影片物件：upload_date（格式 YYYY-MM-DD）
- 前端顯示：upload_date.replace(/-/g, ':') → YYYY:MM:DD

### 影片物件結構（統一格式）
```json
{
  "code": "FC2-PPV-XXXXXXX",
  "title": "影片標題（去掉番號後的乾淨標題）",
  "actress": "女優名稱（或 未知）",
  "cover": "https://封面圖URL",
  "img": "同 cover（我的最愛用 img 欄位）",
  "preview": "https://預覽影片.mp4",
  "url": "https://影片頁面URL",
  "upload_date": "2026-05-29",
  "relative_time": "加入日期: 2026-05-29",
  "type": "無碼破解版 / 正常版 / FC2",
  "found": true,
  "source": "tktube / javxx / tcav / missav"
}
```

### Context Menu 邏輯
- currentContextMenuVideo：當前右鍵圖卡的影片物件
- currentContextMenuSource：'home' 或 'fav'
- source = 'home'：顯示「新增最愛」、隱藏「移除最愛」
- source = 'fav'：顯示「移除最愛」、隱藏「新增最愛」
