## 檔案對照表

| 原始檔案 | main/ 對應 | 說明 |
|---------|-----------|------|
| `app.py` | `japp.py` | 主後端，路徑調整：version_manager import、restart 路徑 |
| `index.html` | `index.html` | 前端主頁，直接複製 |
| `script.js` | `script.js` | 前端邏輯，直接複製 |
| `style.css` | `style.css` | 樣式，直接複製 |
| `version_manager.py` | `version_manager.py` | 版本管理，直接複製 |
| `JApp.spec` | `JApp.spec` | 打包設定，entry point 改為 `japp.py` |
| `fetch_jav.py` | 不複製 | 獨立工具腳本，含硬寫路徑，不屬於主程式 |
| `Favorites.json` | 不複製 | 使用者資料，執行時自動在 main/ 建立 |

---

## 注意事項
- `main/` 的 `Favorites.json` 會在 `main/japp.py` 首次執行後自動建立，與原資料夾隔離
- 打包時在 `main/` 內執行 `pyinstaller JApp.spec` 即可
- 原 `app.py`、`index.html`、`script.js` 等檔案完全不動，作為備份
